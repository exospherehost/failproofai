Хуки и политики для надёжной и автономной работы AI-агентов — для **Claude Code** и **Agents SDK**.

AI-агенты отказывают предсказуемыми способами. Они выполняют деструктивные команды, утекают секреты, отклоняются от задачи, зависают в циклах или делают прямой пуш в main. Без присмотра небольшие сбои превращаются в цепную реакцию.

Failproof AI решает эту проблему с помощью **политик** — правил, которые перехватывают каждый вызов инструмента агентом и могут **обнаруживать сбои**, **устранять их** (блокировать, инструктировать, санировать) и **уведомлять вас**, когда требуется внимание. Локальный дашборд позволяет просматривать всё произошедшее по факту.

Всё работает локально. Никакие данные не покидают вашу машину.

## Быстрый старт

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # включить политики
failproofai                      # запустить дашборд
```

Полное руководство смотрите в разделе [Начало работы](/ru/getting-started).
