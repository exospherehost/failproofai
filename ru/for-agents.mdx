Добавьте полный справочник Failproof AI в ваш агент для программирования одной командой. Работает с Claude Code, Cursor, Windsurf и любым другим агентом, поддерживающим навыки.

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills` определяет, какие агенты у вас установлены, и автоматически добавляет навык в подходящем формате для каждого из них.

## Что охватывает навык

| Область | Что включено |
|---------|--------------|
| Политики | Встроенные имена политик, типы событий, параметры, включение/отключение |
| Пользовательские политики | `customPolicies.add()`, фильтры совпадений, API `allow`/`deny`/`instruct` |
| Объект контекста | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| Конфигурация | Структура `policies-config.json`, слияние областей, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, области видимости |
| Панель управления | Просмотр сессий, активность политик, переменные окружения |
| Архитектура | Поток обработчика хуков, коды выхода, контракт stdin/stdout |

## Навык полный?

Mintlify генерирует `llms.txt` из всех страниц навигации. Документация Failproof AI охватывает полный API — каждая политика, опция и пример включены. Если вы обнаружите что-то отсутствующее, источник находится по адресу `https://docs.befailproof.ai/llms-full.txt`.

Для целенаправленного контекста укажите прямую ссылку на конкретную страницу:

```bash
# Только API пользовательских политик
npx skills add https://docs.befailproof.ai/custom-policies

# Только встроенные политики
npx skills add https://docs.befailproof.ai/built-in-policies
```
