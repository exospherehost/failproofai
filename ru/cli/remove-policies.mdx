---
title: Удаление политик
description: "Удаление записей хуков из настроек Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

Удаляет записи хуков failproofai из файла `settings.json` Claude Code.

Псевдонимы: `failproofai p -u`

## Параметры

| Флаг | Описание |
|------|----------|
| `--scope user` | Удалить из глобальных настроек (по умолчанию) |
| `--scope project` | Удалить из настроек проекта |
| `--scope local` | Удалить из локальных настроек |
| `--scope all` | Удалить из всех областей видимости одновременно |
| `--custom` / `-c` | Очистить `customPoliciesPath` из конфигурации |
| `--beta` | Удалить из конфигурации только бета-политики |

## Поведение

- **Без имён политик** — удаляет все записи хуков failproofai из файла настроек
- **Конкретные имена** — отключает указанные политики, но оставляет хуки установленными

## Примеры

```bash
# Удалить все хуки глобально
failproofai policies --uninstall

# Отключить конкретную политику (хуки остаются установленными)
failproofai policies --uninstall block-sudo

# Удалить хуки из всех областей видимости
failproofai policies --uninstall --scope all

# Очистить путь к пользовательским политикам
failproofai policies --uninstall --custom
```
