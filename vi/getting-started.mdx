## Yêu cầu

- **Node.js** >= 20.9.0
- **Bun** >= 1.3.0 (tùy chọn - chỉ cần thiết khi build từ mã nguồn)

---

## Cài đặt

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

---

## Bắt đầu nhanh

<Steps>
  <Step title="Bật các chính sách">
    Chính sách là các quy tắc chạy trước và sau mỗi lần agent gọi tool. Chúng ngăn chặn các lệnh phá hoại, rò rỉ bí mật và các sự cố khác trước khi gây ra thiệt hại.

    ```bash
    failproofai policies --install
    ```

    Lệnh này ghi các mục hook vào `settings.json` của Claude Code. Bạn cũng có thể cài đặt cho một dự án riêng lẻ hoặc chọn các chính sách cụ thể:

    ```bash
    failproofai policies --install --scope project
    failproofai policies --install block-sudo block-rm-rf sanitize-api-keys
    ```
  </Step>
  <Step title="Xác minh">
    ```bash
    failproofai policies
    ```

    Hiển thị tất cả các chính sách, trạng thái bật/tắt và các tham số đã được cấu hình.
  </Step>
  <Step title="Khởi động dashboard">
    ```bash
    failproofai
    ```

    Mở dashboard cục bộ tại `http://localhost:8020` nơi bạn có thể duyệt các phiên làm việc, kiểm tra các lần gọi tool và quản lý chính sách.
  </Step>
  <Step title="Chạy agent của bạn">
    Khởi động Claude Code như thông thường. Nếu agent thực hiện điều gì đó rủi ro, failproofai sẽ tự động chặn lại. Để chương trình chạy không giám sát và xem lại những gì đã xảy ra trên dashboard.
  </Step>
</Steps>

## Cách chính sách hoạt động

Mỗi khi agent chạy một tool, Claude Code gọi failproofai như một subprocess:

```text
Claude Code  →  failproofai --hook PreToolUse  →  reads stdin JSON
                                                 evaluates policies
                                                 writes decision to stdout
```

Mỗi chính sách trả về một trong ba quyết định:

- **allow** - agent tiếp tục hoạt động bình thường
- **deny** - hành động bị chặn, agent được thông báo lý do
- **instruct** - ngữ cảnh bổ sung được thêm vào prompt của agent

<Note>
Các chính sách chạy trong tiến trình cục bộ của bạn. Không có dữ liệu nào được gửi đến dịch vụ từ xa.
</Note>

## Lưu trữ dữ liệu

Toàn bộ cấu hình và nhật ký được lưu trên máy của bạn:

| Đường dẫn | Nội dung lưu trữ |
|------|----------------|
| `~/.failproofai/policies-config.json` | Cấu hình chính sách toàn cục |
| `~/.failproofai/hook-activity.jsonl` | Lịch sử thực thi hook |
| `~/.failproofai/hook.log` | Nhật ký debug cho lỗi hook tùy chỉnh |
| `.failproofai/policies-config.json` | Cấu hình theo dự án (được commit) |
| `.failproofai/policies-config.local.json` | Ghi đè cá nhân (gitignored) |

---

## Gỡ cài đặt

```bash
failproofai policies --uninstall
```

Xóa các mục hook khỏi `~/.claude/settings.json`. Các tệp cấu hình trong `~/.failproofai/` vẫn được giữ lại.
