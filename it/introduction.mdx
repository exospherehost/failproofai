Hook e policy che mantengono i tuoi agenti AI affidabili e in esecuzione autonoma - per **Claude Code** e l'**Agents SDK**.

Gli agenti AI falliscono in modi prevedibili. Eseguono comandi distruttivi, espongono segreti, si perdono in attività fuori contesto, si bloccano in loop o pubblicano direttamente su main. Se lasciati incustoditi, i piccoli fallimenti si amplificano a cascata.

Failproof AI risolve questo problema con le **policy** - regole che si agganciano a ogni chiamata di strumento dell'agente e possono **rilevare i fallimenti**, **mitigarli** (bloccare, istruire, sanificare) e **avvisarti** quando qualcosa richiede attenzione. Una dashboard locale ti permette di esaminare tutto in seguito.

Tutto gira in locale. Nessun dato lascia il tuo computer.

## Avvio rapido

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # abilita le policy
failproofai                      # avvia la dashboard
```
