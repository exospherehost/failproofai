Hooks ומדיניות שמשמרים את סוכני ה-AI שלך אמינים ופועלים באופן אוטונומי - עבור **Claude Code** ו-**Agents SDK**.

סוכני AI נכשלים בדרכים צפויות. הם מריצים פקודות הרסניות, מדליפים סודות, סוטים מהמשימה, נתקעים בלולאות, או מבצעים push ישירות ל-main. ללא השגחה, כשלים קטנים מסלימים.

Failproof AI פותר זאת באמצעות **policies** - כללים שמתחברים לכל קריאת כלי של הסוכן ויכולים **לזהות כשלים**, **לצמצם אותם** (חסימה, הדרכה, ניקוי), ו**להתריע** כאשר משהו מצריך תשומת לב. לוח מחוונים מקומי מאפשר לך לסקור הכל לאחר מכן.

הכל פועל באופן מקומי. אף מידע לא יוצא מהמחשב שלך.

## התחלה מהירה

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```
