הוסף את כל תיעוד Failproof AI לסוכן הקידוד שלך בפקודה אחת. עובד עם Claude Code, Cursor, Windsurf וכל סוכן אחר שתומך ב-skills.

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills` מזהה אילו סוכנים מותקנים אצלך ומוסיף את ה-skill בפורמט המתאים לכל אחד מהם באופן אוטומטי.

## מה ה-skill מכסה

| תחום | מה כלול |
|------|----------------|
| Policies | שמות מדיניות מובנים, סוגי אירועים, פרמטרים, הפעלה/השבתה |
| Custom policies | `customPolicies.add()`, פילטרי התאמה, ה-API של `allow`/`deny`/`instruct` |
| אובייקט הקשר | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| הגדרות | מבנה `policies-config.json`, מיזוג scopes, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, scopes |
| לוח בקרה | צופה session, פעילות מדיניות, משתני סביבה |
| ארכיטקטורה | זרימת hook handler, קודי יציאה, חוזה stdin/stdout |

## האם ה-skill מלא?

Mintlify מייצר `llms.txt` מכל הדפים בניווט. תיעוד Failproof AI מכסה את ה-API המלא — כל מדיניות, אפשרות ודוגמה כלולים. אם תמצא משהו חסר, המקור נמצא בכתובת `https://docs.befailproof.ai/llms-full.txt`.

לקונטקסט ממוקד, ניתן לקשר ישירות לדף ספציפי:

```bash
# רק ה-API של custom policies
npx skills add https://docs.befailproof.ai/custom-policies

# רק המדיניות המובנות
npx skills add https://docs.befailproof.ai/built-in-policies
```
