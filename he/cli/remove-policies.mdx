---
title: הסרת מדיניות
description: "הסרת רשומות hook מהגדרות Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

מסיר רשומות hook של failproofai מקובץ `settings.json` של Claude Code.

כינויים: `failproofai p -u`

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--scope user` | הסרה מהגדרות גלובליות (ברירת מחדל) |
| `--scope project` | הסרה מהגדרות הפרויקט |
| `--scope local` | הסרה מהגדרות מקומיות |
| `--scope all` | הסרה מכל ההיקפים בבת אחת |
| `--custom` / `-c` | מנקה את `customPoliciesPath` מהתצורה |
| `--beta` | מסיר רק מדיניות בטא מהתצורה |

## התנהגות

- **ללא שמות מדיניות** — מסיר את כל רשומות ה-hook של failproofai מקובץ ההגדרות
- **שמות ספציפיים** — משבית את המדיניות הנבחרות אך משאיר את ה-hooks מותקנים

## דוגמאות

```bash
# הסרת כל ה-hooks ברמה גלובלית
failproofai policies --uninstall

# השבתת מדיניות ספציפית (משאיר את ה-hooks מותקנים)
failproofai policies --uninstall block-sudo

# הסרת hooks מכל ההיקפים
failproofai policies --uninstall --scope all

# ניקוי נתיב מדיניות מותאם אישית
failproofai policies --uninstall --custom
```
