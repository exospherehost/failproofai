---
title: צפייה בסשנים
description: "הפעל את לוח הבקרה לעיון בסשני הסוכן וניהול מדיניות"
---

```bash
failproofai
```

מפעיל את לוח הבקרה הווב בכתובת `http://localhost:8020`.

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--port <number>` | פורט להאזנה (ברירת מחדל: `8020`) |
| `--projects-path <path>` | עקיפת המיקום שבו נמצאות תיקיות הפרויקטים של Claude Code |
| `--allowed-origins <origins>` | מארחים/כתובות IP מופרדות בפסיקים שמורשים לגשת למשאבי פיתוח |

## דוגמאות

```bash
# הפעלה על פורט שונה
failproofai --port 9000

# שימוש בנתיב פרויקטים מותאם אישית
failproofai --projects-path ~/my-claude-projects
```
