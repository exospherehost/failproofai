title: Ortam değişkenleri
description: "failproofai davranışını ortam değişkenleriyle yapılandırın"
---

## Dashboard

| Değişken | Açıklama |
|----------|-------------|
| `PORT` | Dashboard portu (varsayılan: `8020`) |
| `CLAUDE_PROJECTS_PATH` | Claude Code proje klasörlerinin bulunduğu konumu geçersiz kılar |
| `FAILPROOFAI_DISABLE_PAGES=policies,projects` | Gizlenecek dashboard sayfalarının virgülle ayrılmış listesi |
| `FAILPROOFAI_ALLOWED_DEV_ORIGINS` | Geliştirici kaynaklarına erişmesine izin verilen host'lar/IP'ler. `--allowed-origins` ile aynıdır. |

## Günlükleme

| Değişken | Açıklama |
|----------|-------------|
| `FAILPROOFAI_LOG_LEVEL=info\|warn\|error` | Sunucu günlük düzeyi (varsayılan: `warn`) |
| `FAILPROOFAI_HOOK_LOG_FILE` | Özel hook günlük dosyası yolu veya varsayılan için `true` (`~/.failproofai/logs/hooks.log`) |

## Telemetri

| Değişken | Açıklama |
|----------|-------------|
| `FAILPROOFAI_TELEMETRY_DISABLED=1` | Anonim kullanım telemetrisini devre dışı bırakır |

## LLM (politika değerlendirmesi için)

| Değişken | Açıklama |
|----------|-------------|
| `FAILPROOFAI_LLM_BASE_URL` | LLM API uç noktası (varsayılan: `https://api.openai.com/v1`) |
| `FAILPROOFAI_LLM_API_KEY` | LLM destekli politikalar için API anahtarı |
| `FAILPROOFAI_LLM_MODEL` | Model adı (varsayılan: `gpt-4o-mini`) |
