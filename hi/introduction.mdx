Hooks और policies जो आपके AI एजेंट्स को विश्वसनीय और स्वायत्त रूप से चलाते रहते हैं - **Claude Code** और **Agents SDK** के लिए।

AI एजेंट्स अनुमानित तरीकों से विफल होते हैं। वे विनाशकारी कमांड चलाते हैं, secrets लीक करते हैं, काम से भटक जाते हैं, loops में फँस जाते हैं, या सीधे main में push कर देते हैं। बिना निगरानी के, छोटी विफलताएँ बड़ी समस्या बन जाती हैं।

Failproof AI इसे **policies** से हल करता है - ऐसे नियम जो हर एजेंट tool call में hook होते हैं और **विफलताओं का पता लगा** सकते हैं, **उन्हें रोक** सकते हैं (block, instruct, sanitize), और जब कुछ ध्यान देने की ज़रूरत हो तो **आपको सूचित** कर सकते हैं। एक लोकल डैशबोर्ड आपको बाद में सब कुछ समीक्षा करने देता है।

सब कुछ लोकल रूप से चलता है। कोई भी डेटा आपकी मशीन से बाहर नहीं जाता।

## त्वरित शुरुआत

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # policies सक्षम करें
failproofai                      # डैशबोर्ड लॉन्च करें
```

पूरी जानकारी के लिए [Getting started](/hi/getting-started) गाइड देखें।
