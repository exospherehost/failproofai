---

## Dashboard

| Variable | Description |
|----------|-------------|
| `PORT` | Dashboard पोर्ट (डिफ़ॉल्ट: `8020`) |
| `CLAUDE_PROJECTS_PATH` | Claude Code प्रोजेक्ट फ़ोल्डर की लोकेशन को ओवरराइड करें |
| `FAILPROOFAI_DISABLE_PAGES=policies,projects` | छुपाने के लिए Dashboard पेजों की कॉमा-separated सूची |
| `FAILPROOFAI_ALLOWED_DEV_ORIGINS` | Dev रिसोर्स तक एक्सेस करने की अनुमति प्राप्त Hosts/IPs। `--allowed-origins` के समान। |

## Logging

| Variable | Description |
|----------|-------------|
| `FAILPROOFAI_LOG_LEVEL=info\|warn\|error` | सर्वर लॉग लेवल (डिफ़ॉल्ट: `warn`) |
| `FAILPROOFAI_HOOK_LOG_FILE` | कस्टम hook लॉग फ़ाइल पाथ, या डिफ़ॉल्ट के लिए `true` (`~/.failproofai/logs/hooks.log`) |

## Telemetry

| Variable | Description |
|----------|-------------|
| `FAILPROOFAI_TELEMETRY_DISABLED=1` | अनाम उपयोग टेलीमेट्री अक्षम करें |

## LLM (policy मूल्यांकन के लिए)

| Variable | Description |
|----------|-------------|
| `FAILPROOFAI_LLM_BASE_URL` | LLM API एंडपॉइंट (डिफ़ॉल्ट: `https://api.openai.com/v1`) |
| `FAILPROOFAI_LLM_API_KEY` | LLM-संचालित policies के लिए API key |
| `FAILPROOFAI_LLM_MODEL` | मॉडल का नाम (डिफ़ॉल्ट: `gpt-4o-mini`) |
