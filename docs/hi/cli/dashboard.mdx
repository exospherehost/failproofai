```bash
failproofai
```

`http://localhost:8020` पर वेब डैशबोर्ड शुरू करता है।

## विकल्प

| फ्लैग | विवरण |
|------|-------------|
| `--port <number>` | सुनने के लिए पोर्ट (डिफ़ॉल्ट: `8020`) |
| `--allowed-origins <origins>` | अल्पविराम-सीमांकित होस्ट/आईपी जो डेव संसाधनों तक पहुंचने की अनुमति हैं |

डैशबोर्ड को गैर-डिफ़ॉल्ट Claude प्रोजेक्ट फ़ोल्डर पर इंगित करने के लिए, लॉन्च करते समय `CLAUDE_PROJECTS_PATH` पर्यावरण चर सेट करें।

## उदाहरण

```bash
# एक अलग पोर्ट पर लॉन्च करें
failproofai --port 9000

# पर्यावरण चर के माध्यम से कस्टम Claude प्रोजेक्ट पथ का उपयोग करें
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
