हुकस और पॉलिसीज़ जो आपके AI एजेंटों को विश्वसनीय रखते हैं और स्वायत्त रूप से चलाते हैं - **Claude Code** और **Agents SDK** के लिए।

AI एजेंट अनुमानित तरीकों से विफल होते हैं। वे विनाशकारी कमांड चलाते हैं, राज़ सिकोड़ते हैं, कार्य से भटकते हैं, लूप में फंसते हैं, या सीधे मुख्य शाखा में पुश करते हैं। अनुचर छोड़ने पर, छोटी विफलताएं बड़ी हो जाती हैं।

Failproof AI इसे **पॉलिसीज़** के साथ हल करता है - नियम जो प्रत्येक एजेंट टूल कॉल में हुक करते हैं और **विफलताओं का पता लगा** सकते हैं, **उन्हें कम कर** सकते हैं (ब्लॉक करना, निर्देश देना, साफ़ करना), और **आपको सतर्क कर** सकते हैं जब कुछ ध्यान की जरूरत हो। एक स्थानीय डैशबोर्ड आपको बाद में सब कुछ समीक्षा करने देता है।

सब कुछ स्थानीय रूप से चलता है। कोई डेटा आपकी मशीन से बाहर नहीं जाता।

## त्वरित शुरुआत

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # पॉलिसीज़ सक्षम करें
failproofai                      # डैशबोर्ड लॉन्च करें
```
