Hooks और policies जो आपके AI agents को विश्वसनीय और स्वायत्त रूप से चलाते रहते हैं - **Claude Code**, **OpenAI Codex**, और **Agents SDK** के लिए।

AI agents अनुमानित तरीकों से विफल हो जाते हैं। वे विनाशकारी commands चलाते हैं, secrets को leak करते हैं, कार्य से भटक जाते हैं, loops में फंस जाते हैं, या सीधे main में push कर देते हैं। बिना निरीक्षण के, छोटी विफलताएं बढ़ जाती हैं।

Failproof AI इसे **policies** के साथ हल करता है - नियम जो हर agent tool call में hook करते हैं और **विफलताओं का पता लगा** सकते हैं, **उन्हें कम कर** सकते हैं (block करें, instruct दें, sanitize करें), और **जब कुछ ध्यान की जरूरत हो तो आपको alert** कर सकते हैं। एक local dashboard आपको बाद में सबकुछ review करने देता है।

सबकुछ locally चलता है। आपकी machine से कोई डेटा बाहर नहीं जाता।

## क्विक स्टार्ट

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```
