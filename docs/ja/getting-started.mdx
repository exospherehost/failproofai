## 必要要件

- **Node.js** >= 20.9.0
- **Bun** >= 1.3.0（オプション — ソースからビルドする場合のみ必要）

---

## インストール

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

---

## クイックスタート

<Steps>
  <Step title="ポリシーを有効化する">
    ポリシーは、エージェントのツール呼び出しの前後に実行されるルールです。破壊的なコマンド、シークレットの漏洩、その他の障害パターンを、実害が発生する前に検出します。

    ```bash
    failproofai policies --install
    ```

    このコマンドは Claude Code の `settings.json` にフックエントリを書き込みます。単一プロジェクト向けにインストールしたり、特定のポリシーを選択したりすることもできます：

    ```bash
    failproofai policies --install --scope project
    failproofai policies --install block-sudo block-rm-rf sanitize-api-keys
    ```
  </Step>
  <Step title="確認する">
    ```bash
    failproofai policies
    ```

    すべてのポリシーの一覧と、それぞれの有効／無効の状態、設定済みパラメータが表示されます。
  </Step>
  <Step title="ダッシュボードを起動する">
    ```bash
    failproofai
    ```

    `http://localhost:8020` でローカルダッシュボードが開き、セッションの閲覧、ツール呼び出しの確認、ポリシーの管理が行えます。
  </Step>
  <Step title="エージェントを実行する">
    Claude Code を通常どおり起動します。エージェントがリスクのある操作を試みると、failproofai が自動的に介入します。そのまま無人で実行させておき、ダッシュボードで後から内容を確認できます。
  </Step>
</Steps>

---

## ポリシーの仕組み

エージェントがツールを実行するたびに、Claude Code はサブプロセスとして failproofai を呼び出します：

```text
Claude Code  →  failproofai --hook PreToolUse  →  reads stdin JSON
                                                 evaluates policies
                                                 writes decision to stdout
```

各ポリシーは以下の3つのいずれかを返します：

- **allow** — エージェントはそのまま処理を続行する
- **deny** — アクションがブロックされ、エージェントに理由が通知される
- **instruct** — エージェントのプロンプトに追加のコンテキストが付与される

<Note>
ポリシーはローカルプロセス内で実行されます。リモートサービスには何も送信されません。
</Note>

## データの保存場所

すべての設定とログはローカルマシン上に保存されます：

| パス | 保存内容 |
|------|----------------|
| `~/.failproofai/policies-config.json` | グローバルポリシー設定 |
| `~/.failproofai/hook-activity.jsonl` | フック実行履歴 |
| `~/.failproofai/hook.log` | カスタムフックエラーのデバッグログ |
| `.failproofai/policies-config.json` | プロジェクト固有の設定（コミット対象） |
| `.failproofai/policies-config.local.json` | 個人用オーバーライド（gitignore対象） |

---

## アンインストール

```bash
failproofai policies --uninstall
```

`~/.claude/settings.json` からフックエントリを削除します。`~/.failproofai/` 内の設定ファイルは保持されます。
