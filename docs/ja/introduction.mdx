**Claude Code**、**OpenAI Codex**、**Agents SDK** 向けに、AIエージェントの安定性と自律稼働を支えるフックとポリシーを提供します。

AIエージェントは、予測可能な形で失敗します。破壊的なコマンドの実行、シークレットの漏洩、タスクからの逸脱、ループへのはまり込み、mainブランチへの直接プッシュなど、さまざまな問題が起きます。放置すると、小さな失敗が連鎖的に拡大します。

Failproof AI はこの課題を**ポリシー**で解決します。ポリシーはエージェントのすべてのツール呼び出しにフックし、**障害を検出**し、**対処**（ブロック、指示、サニタイズ）し、注意が必要な場合に**通知**します。ローカルダッシュボードで、後からすべての履歴を確認できます。

すべてはローカルで動作します。データが外部に送信されることはありません。

## クイックスタート

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # ポリシーを有効化
failproofai                      # ダッシュボードを起動
```

詳細な手順については、[はじめかた](/ja/getting-started)ガイドをご覧ください。
