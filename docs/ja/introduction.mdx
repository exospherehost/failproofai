AIエージェントの信頼性を保ち、自律的な実行を維持するフックとポリシー — **Claude Code**、**OpenAI Codex**、**Agents SDK** に対応。

AIエージェントは予測可能な方法で失敗します。破壊的なコマンドを実行したり、シークレットを漏洩したり、タスクから逸脱したり、ループに陥ったり、mainブランチに直接プッシュしたりします。放置すると、小さな失敗が連鎖的に拡大していきます。

Failproof AI はこの問題を**ポリシー**で解決します。ポリシーとは、エージェントのすべてのツール呼び出しにフックするルールであり、**障害の検知**、**軽減処置**（ブロック、指示、サニタイズ）、そして**注意が必要な際のアラート通知**を行うことができます。ローカルダッシュボードで、後からすべての内容を確認できます。

すべてはローカルで動作します。データが外部に送信されることはありません。

<Card title="26種類の組み込みポリシー" icon="shield" href="/ja/built-in-policies">

## クイックスタート

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # ポリシーを有効化
failproofai                      # ダッシュボードを起動
```

詳しい手順については、[はじめかた](/ja/getting-started)ガイドを参照してください。
