Hook e policy che mantengono i tuoi agenti AI affidabili e in esecuzione autonoma - per **Claude Code**, **OpenAI Codex** e **Agents SDK**.

Gli agenti AI falliscono in modi prevedibili. Eseguono comandi distruttivi, perdono segreti, si allontanano dal compito assegnato, rimangono bloccati in loop, o eseguono il push direttamente al main. Se lasciati incustoditi, i piccoli errori si cascano.

Failproof AI risolve questo con le **policy** - regole che si agganciano a ogni chiamata di strumento dell'agente e possono **rilevare gli errori**, **mitigarli** (bloccare, istruire, disinfettare) e **avvertirti** quando qualcosa ha bisogno di attenzione. Una dashboard locale ti permette di rivedere tutto successivamente.

Tutto funziona localmente. Nessun dato lascia la tua macchina.

## Avvio rapido

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # abilita le policy
failproofai                      # avvia la dashboard
```

Vedi la guida [Inizia](/it/getting-started) per la procedura dettagliata completa.
