---
title: הסרת מדיניויות
description: "הסרת רשומות hook מהגדרות Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

מסירה רשומות failproofai hook מ-`settings.json` של Claude Code.

כינויים: `failproofai p -u`

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--scope user` | הסרה מהגדרות גלובליות (ברירת מחדל) |
| `--scope project` | הסרה מהגדרות פרויקט |
| `--scope local` | הסרה מהגדרות מקומיות |
| `--scope all` | הסרה מכל ההיקפים בבת אחת |
| `--custom` / `-c` | ניקוי של `customPoliciesPath` מהתצורה |
| `--beta` | הסרה של מדיניויות בטא בלבד מהתצורה |

## התנהגות

- **ללא שמות מדיניויות** - מסירה את כל רשומות failproofai hook מקובץ ההגדרות
- **שמות ספציפיים** - מכביא מדיניויות אלה אך משאיר hooks מותקנים

## דוגמאות

```bash
# הסרת כל hooks באופן גלובלי
failproofai policies --uninstall

# השבתת מדיניות ספציפית (שומרת hooks מותקנים)
failproofai policies --uninstall block-sudo

# הסרת hooks מכל היקף
failproofai policies --uninstall --scope all

# ניקוי נתיב מדיניויות מותאם
failproofai policies --uninstall --custom
```
