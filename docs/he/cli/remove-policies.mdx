---
title: הסרת מדיניויות
description: "הסר רשומות hook מהגדרות Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

מסיר רשומות failproofai hook מ-`settings.json` של Claude Code.

כינויים: `failproofai p -u`

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--scope user` | הסר מהגדרות גלובליות (ברירת מחדל) |
| `--scope project` | הסר מהגדרות הפרויקט |
| `--scope local` | הסר מהגדרות מקומיות |
| `--scope all` | הסר מכל ההיקפים בבת אחת |
| `--custom` / `-c` | נקה את ה-`customPoliciesPath` מהתצורה |

## התנהגות

- **ללא שמות מדיניויות** - מסיר את כל רשומות failproofai hook מקובץ ההגדרות
- **שמות ספציפיים** - משבית מדיניויות אלו אך שומרת hooks מותקנים

## דוגמאות

```bash
# הסר את כל ה-hooks באופן גלובלי
failproofai policies --uninstall

# השבת מדיניות ספציפית (שומרת hooks מותקנים)
failproofai policies --uninstall block-sudo

# הסר hooks מכל היקף
failproofai policies --uninstall --scope all

# נקה את נתיב המדיניויות המותאמות
failproofai policies --uninstall --custom
```
