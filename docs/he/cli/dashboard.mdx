---
title: צפייה בהפעלות
description: "הפעל את לוח הבקרה לעיון בהפעלות סוכנים וניהול מדיניות"
---

```bash
failproofai
```

מפעיל את לוח הבקרה באתר `http://localhost:8020`.

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--port <number>` | הפורט להאזנה אליו (ברירת מחדל: `8020`) |
| `--allowed-origins <origins>` | רשימה מופרדת בפסיקים של שמות משתמן/כתובות IP המורשות לגישה למשאבי הפיתוח |

כדי להצביע את לוח הבקרה לתיקיית Claude project שאינה ברירת המחדל, הגדר את משתנה הסביבה `CLAUDE_PROJECTS_PATH` בעת ההפעלה.

## דוגמאות

```bash
# הפעל בפורט שונה
failproofai --port 9000

# השתמש בנתיב Claude projects מותאם אישית דרך משתנה סביבה
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
