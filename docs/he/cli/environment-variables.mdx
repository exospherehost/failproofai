## לוח הבקרה

| משתנה | תיאור |
|----------|-------------|
| `PORT` | פורט לוח הבקרה (ברירת מחדל: `8020`) |
| `CLAUDE_PROJECTS_PATH` | עקיפת המיקום שבו נמצאות תיקיות הפרויקטים של Claude Code |
| `FAILPROOFAI_DISABLE_PAGES=policies,projects` | דפי לוח הבקרה להסתרה, מופרדים בפסיקים |
| `FAILPROOFAI_ALLOWED_DEV_ORIGINS` | מארחים/כתובות IP שמורשים לגשת למשאבי פיתוח. זהה ל-`--allowed-origins`. |

## תיעוד (Logging)

| משתנה | תיאור |
|----------|-------------|
| `FAILPROOFAI_LOG_LEVEL=info\|warn\|error` | רמת תיעוד השרת (ברירת מחדל: `warn`) |
| `FAILPROOFAI_HOOK_LOG_FILE` | נתיב מותאם אישית לקובץ יומן ה-hook, או `true` לנתיב ברירת המחדל (`~/.failproofai/logs/hooks.log`) |

## טלמטריה

| משתנה | תיאור |
|----------|-------------|
| `FAILPROOFAI_TELEMETRY_DISABLED=1` | השבתת איסוף נתוני שימוש אנונימיים |

## LLM (להערכת מדיניות)

| משתנה | תיאור |
|----------|-------------|
| `FAILPROOFAI_LLM_BASE_URL` | נקודת הקצה של ה-LLM API (ברירת מחדל: `https://api.openai.com/v1`) |
| `FAILPROOFAI_LLM_API_KEY` | מפתח API עבור מדיניות מבוססת LLM |
| `FAILPROOFAI_LLM_MODEL` | שם המודל (ברירת מחדל: `gpt-4o-mini`) |
