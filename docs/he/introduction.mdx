Hooks ו-policies שמשמרים את AI agents שלך אמינים וריצים באופן עצמאי - עבור **Claude Code**, **OpenAI Codex**, וה-**Agents SDK**.

AI agents נכשלים בדרכים צפויות. הם מריצים פקודות הרסניות, דולפים סודות, סוטים מהמשימה, נתקעים בלולאות, או דוחפים ישירות ל-main. אם משאירים אותם ללא פיקוח, כשלים קטנים מתגברים.

Failproof AI פותר זאת עם **policies** - כללים שחוברים לכל קריאת tool של agent ויכולים **לזהות כשלים**, **להפחית אותם** (לחסום, להנחות, לנקות), ו**להתריע עליך** כשמשהו זקוק לתשומת לב. דשבורד מקומי מאפשר לך לבדוק הכל לאחר מכן.

הכל רץ באופן מקומי. אף נתונים לא עוזבים את המכונה שלך.

## התחלה מהירה

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```
