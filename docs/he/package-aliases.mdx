## החבילה הרשמית

חבילת npm הקנונית היא **`failproofai`**:

```bash
npm install -g failproofai
# or
bun add -g failproofai
```

---

## למה אנחנו בעלים של שמות הכינויים

Typosquatting הוא התקפת supply-chain נפוצה שבה שחקן זדוני רושם שם חבילה שהוא דקה אחת מהקלדה מחבילה פופולרית. משתמשים שלא משגיחים שטעו בפקודת ההתקנה בסופו של דבר מריצים קוד מבוקר על ידי התוקף עם גישה מלאה למערכת - בדיוק סוג האיום שFailproof AI תוכנן להגן עליו.

כדי למחוק את המשטח הזה, **אנחנו בבעלות קדימוני של כל ההקלדות והווריאציות של עיצוב שגויים נפוצות** של `failproofai` ב-npm. שום אחד משמות אלה לא יכול להיות רשום על ידי צד שלישי. כל אחד מהם הוא proxy דק המתקין ומעביר למשימה לחבילת `failproofai` האמיתית.

## איך הכינויים עובדים

כל חבילת כינוי:

1. רושמת `failproofai` כתלות - כך שהחבילה האמיתית (כולל התקנת hook ה-`postinstall` שלה) פועלת בהתקנה
2. חושפת בינארי התואם לשמה שלה (למשל `failprof-ai`) שמעביר כל ארגומנטים לבינארי `failproofai`

ה-proxy הוא סקריפט Node בן שתי שורות; אין לוגיקה, אין קריאות רשת, וללא איסוף נתונים מעבר למה ש-`failproofai` עצמו עושה.
