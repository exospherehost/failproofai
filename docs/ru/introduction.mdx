Хуки и политики, которые делают ваших AI-агентов надежными и автономными — для **Claude Code**, **OpenAI Codex** и **Agents SDK**.

AI-агенты отказывают предсказуемым образом. Они запускают деструктивные команды, раскрывают секреты, отвлекаются от задачи, застревают в циклах или пушат напрямую в main. Если не контролировать, небольшие сбои каскадируют.

Failproof AI решает это с помощью **политик** — правил, которые подключаются к каждому вызову инструмента агента и могут **обнаруживать сбои**, **смягчать их** (блокировать, инструктировать, санитизировать) и **уведомлять вас** когда нужно внимание. Локальная панель управления позволяет просмотреть всё впоследствии.

Всё работает локально. Никакие данные не покидают вашу машину.

## Быстрый старт

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # включить политики
failproofai                      # запустить панель управления
```

Полное руководство см. в [Руководстве по началу работы](/ru/getting-started).
