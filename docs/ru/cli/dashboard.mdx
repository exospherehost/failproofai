---
title: Просмотр сессий
description: "Запустите дашборд для просмотра сессий агента и управления политиками"
---

```bash
failproofai
```

Запускает веб-дашборд по адресу `http://localhost:8020`.

## Параметры

| Флаг | Описание |
|------|----------|
| `--port <number>` | Порт для прослушивания (по умолчанию: `8020`) |
| `--projects-path <path>` | Переопределяет путь к папкам проектов Claude Code |
| `--allowed-origins <origins>` | Хосты/IP-адреса через запятую, которым разрешён доступ к ресурсам для разработки |

## Примеры

```bash
# Запуск на другом порту
failproofai --port 9000

# Использование пользовательского пути к проектам
failproofai --projects-path ~/my-claude-projects
```
