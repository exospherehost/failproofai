## Панель управления

| Переменная | Описание |
|----------|-------------|
| `PORT` | Порт панели управления (по умолчанию: `8020`) |
| `CLAUDE_PROJECTS_PATH` | Переопределяет путь к папкам проектов Claude Code |
| `FAILPROOFAI_DISABLE_PAGES=policies,projects` | Страницы панели управления, разделённые запятыми, которые нужно скрыть |
| `FAILPROOFAI_ALLOWED_DEV_ORIGINS` | Хосты/IP-адреса, которым разрешён доступ к ресурсам разработки. Аналог `--allowed-origins`. |

## Логирование

| Переменная | Описание |
|----------|-------------|
| `FAILPROOFAI_LOG_LEVEL=info\|warn\|error` | Уровень логирования сервера (по умолчанию: `warn`) |
| `FAILPROOFAI_HOOK_LOG_FILE` | Путь к пользовательскому файлу логов хуков или `true` для использования пути по умолчанию (`~/.failproofai/logs/hooks.log`) |

## Телеметрия

| Переменная | Описание |
|----------|-------------|
| `FAILPROOFAI_TELEMETRY_DISABLED=1` | Отключить анонимную телеметрию использования |

## LLM (для оценки политик)

| Переменная | Описание |
|----------|-------------|
| `FAILPROOFAI_LLM_BASE_URL` | Конечная точка API LLM (по умолчанию: `https://api.openai.com/v1`) |
| `FAILPROOFAI_LLM_API_KEY` | API-ключ для политик на основе LLM |
| `FAILPROOFAI_LLM_MODEL` | Название модели (по умолчанию: `gpt-4o-mini`) |
