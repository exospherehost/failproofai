---
title: Удаление политик
description: "Удаление записей hook из настроек Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

Удаляет записи failproofai hook из `settings.json` Claude Code.

Синонимы: `failproofai p -u`

## Параметры

| Флаг | Описание |
|------|----------|
| `--scope user` | Удалить из глобальных настроек (по умолчанию) |
| `--scope project` | Удалить из настроек проекта |
| `--scope local` | Удалить из локальных настроек |
| `--scope all` | Удалить из всех областей одновременно |
| `--custom` / `-c` | Очистить `customPoliciesPath` из конфигурации |
| `--beta` | Удалить только бета-политики из конфигурации |

## Поведение

- **Без названий политик** — удаляет все записи failproofai hook из файла настроек
- **Конкретные названия** — отключает указанные политики, но оставляет hooks установленными

## Примеры

```bash
# Удалить все hooks глобально
failproofai policies --uninstall

# Отключить конкретную политику (оставляет hooks установленными)
failproofai policies --uninstall block-sudo

# Удалить hooks из всех областей
failproofai policies --uninstall --scope all

# Очистить путь к пользовательским политикам
failproofai policies --uninstall --custom
```
