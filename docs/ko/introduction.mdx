**Claude Code**, **OpenAI Codex**, **Agents SDK**를 위한 훅과 정책 — AI 에이전트를 안정적으로 자율 실행할 수 있도록 도와줍니다.

AI 에이전트는 예측 가능한 방식으로 실패합니다. 위험한 명령을 실행하거나, 시크릿을 유출하거나, 목표에서 벗어나거나, 루프에 빠지거나, main 브랜치에 직접 푸시하기도 합니다. 방치하면 작은 오류가 연쇄적으로 커집니다.

Failproof AI는 **정책(policies)**으로 이 문제를 해결합니다. 정책은 에이전트의 모든 도구 호출에 훅을 걸어 **장애를 감지**하고, **차단·지시·정제 등의 방식으로 완화**하며, 주의가 필요한 상황을 **알림**으로 전달합니다. 로컬 대시보드에서 이후에 모든 내역을 검토할 수 있습니다.

모든 처리는 로컬에서 이루어집니다. 어떤 데이터도 외부로 전송되지 않습니다.

## 빠른 시작

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # 정책 활성화
failproofai                      # 대시보드 실행
```

전체 안내는 [시작하기](/ko/getting-started) 가이드를 참고하세요.
