---
title: عرض الجلسات
description: "قم بتشغيل لوحة التحكم لتصفح جلسات الوكيل وإدارة السياسات"
---

```bash
failproofai
```

يبدأ لوحة التحكم الويب على `http://localhost:8020`.

## الخيارات

| الراية | الوصف |
|------|-------------|
| `--port <number>` | المنفذ المراد الاستماع عليه (الافتراضي: `8020`) |
| `--projects-path <path>` | تجاوز المسار الذي يتم البحث فيه عن مجلدات مشاريع Claude Code |
| `--allowed-origins <origins>` | المضيفين/عناوين IP المفصولة بفواصل المسموح بها للوصول إلى موارد التطوير |

## أمثلة

```bash
# قم بالتشغيل على منفذ مختلف
failproofai --port 9000

# استخدم مسار مشاريع مخصص
failproofai --projects-path ~/my-claude-projects
```
