---
title: عرض الجلسات
description: "تشغيل لوحة التحكم لتصفح جلسات الوكلاء وإدارة السياسات"
---

```bash
failproofai
```

يشغّل لوحة التحكم على الويب عبر العنوان `http://localhost:8020`.

## الخيارات

| الخيار | الوصف |
|--------|--------|
| `--port <number>` | المنفذ المراد الاستماع عليه (الافتراضي: `8020`) |
| `--projects-path <path>` | تجاوز المسار الذي يتم فيه البحث عن مجلدات مشاريع Claude Code |
| `--allowed-origins <origins>` | قائمة من المضيفين أو عناوين IP مفصولة بفواصل يُسمح لها بالوصول إلى موارد التطوير |

## أمثلة

```bash
# Launch on a different port
failproofai --port 9000

# Use a custom projects path
failproofai --projects-path ~/my-claude-projects
```
