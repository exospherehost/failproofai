---
title: عرض الجلسات
description: "قم بتشغيل لوحة التحكم لاستعراض جلسات الوكيل وإدارة السياسات"
---

```bash
failproofai
```

يبدأ لوحة التحكم الويب على `http://localhost:8020`.

## الخيارات

| العلم | الوصف |
|------|-------------|
| `--port <number>` | المنفذ المراد الاستماع عليه (القيمة الافتراضية: `8020`) |
| `--allowed-origins <origins>` | مضيفين/عناوين IP مفصولة بفواصل مسموحة للوصول إلى موارد التطوير |

لتوجيه لوحة التحكم إلى مجلد مشروع Claude غير افتراضي، قم بتعيين متغير البيئة `CLAUDE_PROJECTS_PATH` عند بدء التشغيل.

## أمثلة

```bash
# قم بالتشغيل على منفذ مختلف
failproofai --port 9000

# استخدم مسار مشاريع Claude مخصص عبر متغير البيئة
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
