خطافات وسياسات تحافظ على موثوقية عملاء الذكاء الاصطناعي الخاصة بك وتبقيهم يعملون بشكل مستقل - لـ **Claude Code** و **OpenAI Codex** و **Agents SDK**.

عملاء الذكاء الاصطناعي يفشلون بطرق يمكن التنبؤ بها. إنهم ينفذون أوامر تخريبية، ويسربون الأسرار، ويحيدون عن المهمة، ويعلقون في حلقات لا نهائية، أو يدفعون مباشرة إلى الفرع الرئيسي. إذا تُرِكت دون مراقبة، فإن الأعطال الصغيرة تتسبب في كوارث متسلسلة.

يحل Failproof AI هذه المشكلة باستخدام **السياسات** - قواعد تتصل بكل استدعاء أداة للعامل وتستطيع **اكتشاف الأعطال**، **تخفيف آثارها** (الحجب والتعليمات والتطهير)، و **تنبيهك** عند الحاجة إلى الانتباه. تتيح لك لوحة معلومات محلية مراجعة كل شيء لاحقًا.

كل شيء يعمل محليًا. لا تغادر البيانات جهازك.

## البدء السريع

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # تفعيل السياسات
failproofai                      # إطلاق لوحة المعلومات
```

اطلع على دليل [البدء السريع](/ar/getting-started) للحصول على الشرح الكامل.
