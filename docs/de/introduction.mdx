Hooks und Richtlinien, die Ihre KI-Agenten zuverlässig und autonom am Laufen halten – für **Claude Code** und das **Agents SDK**.

KI-Agenten scheitern auf vorhersehbare Weise. Sie führen destruktive Befehle aus, geben Geheimnisse preis, verlieren den Fokus, geraten in Schleifen oder pushen direkt in den Hauptzweig. Ohne Aufsicht können kleine Fehler eskalieren.

Failproof AI löst dies mit **Richtlinien** – Regeln, die sich in jeden Agenten-Tool-Aufruf einklinken und **Fehler erkennen**, **abschwächen** (blockieren, anweisen, bereinigen) und **Sie benachrichtigen** können, wenn etwas Aufmerksamkeit erfordert. Ein lokales Dashboard ermöglicht es Ihnen, alles im Nachhinein zu überprüfen.

Alles läuft lokal. Keine Daten verlassen Ihren Rechner.

## Schnellstart

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # Richtlinien aktivieren
failproofai                      # Dashboard starten
```

Den vollständigen Leitfaden finden Sie unter [Erste Schritte](/de/getting-started).
