Hooks und Policies, die deine KI-Agenten zuverlässig und autonom am Laufen halten – für **Claude Code** und das **Agents SDK**.

KI-Agenten scheitern auf vorhersehbare Weise. Sie führen destruktive Befehle aus, geben Secrets preis, verlieren den Fokus, geraten in Endlosschleifen oder pushen direkt auf main. Werden solche Fehler nicht beachtet, können kleine Probleme schnell größere Schäden verursachen.

Failproof AI löst dies mit **Policies** – Regeln, die bei jedem Werkzeugaufruf eines Agenten greifen und Fehler **erkennen**, **beheben** (blockieren, anweisen, bereinigen) und **dich benachrichtigen** können, wenn etwas Aufmerksamkeit erfordert. Ein lokales Dashboard ermöglicht dir die anschließende Überprüfung aller Vorgänge.

Alles läuft lokal. Keine Daten verlassen deinen Rechner.

## Schnellstart

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # Policies aktivieren
failproofai                      # Dashboard starten
```

Eine vollständige Anleitung findest du im [Erste Schritte](/de/getting-started)-Leitfaden.
