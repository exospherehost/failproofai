Hooks und Richtlinien, die Ihre KI-Agenten zuverlässig und autonom am Laufen halten – für **Claude Code**, **OpenAI Codex** und das **Agents SDK**.

KI-Agenten scheitern auf vorhersehbare Weise. Sie führen destruktive Befehle aus, geben Geheimnisse preis, weichen von ihrer Aufgabe ab, geraten in Endlosschleifen oder pushen direkt in den Main-Branch. Ohne Aufsicht können kleine Fehler schnell eskalieren.

Failproof AI löst dies mit **Richtlinien** – Regeln, die sich in jeden Werkzeugaufruf eines Agenten einhaken und Fehler **erkennen**, **beheben** (blockieren, anweisen, bereinigen) und Sie **benachrichtigen** können, wenn etwas Aufmerksamkeit erfordert. Ein lokales Dashboard ermöglicht es Ihnen, alles im Nachhinein zu überprüfen.

Alles läuft lokal. Keine Daten verlassen Ihren Rechner.

## Schnellstart

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # Richtlinien aktivieren
failproofai                      # Dashboard starten
```

Den vollständigen Durchlauf finden Sie im [Erste-Schritte-Leitfaden](/de/getting-started).
