---
title: Sitzungen anzeigen
description: "Dashboard starten, um Agent-Sitzungen zu durchsuchen und Richtlinien zu verwalten"
---

```bash
failproofai
```

Startet das Web-Dashboard unter `http://localhost:8020`.

## Optionen

| Flag | Beschreibung |
|------|--------------|
| `--port <number>` | Port, auf dem gehorcht werden soll (Standard: `8020`) |
| `--allowed-origins <origins>` | Kommagetrennte Hosts/IPs, die auf Dev-Ressourcen zugreifen dürfen |

Um das Dashboard auf einen nicht standardmäßigen Claude-Projektordner auszurichten, setzen Sie beim Starten die Umgebungsvariable `CLAUDE_PROJECTS_PATH`.

## Beispiele

```bash
# Auf einem anderen Port starten
failproofai --port 9000

# Einen benutzerdefinierten Claude-Projektpfad über eine Umgebungsvariable verwenden
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
