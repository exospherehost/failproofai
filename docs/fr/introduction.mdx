Des hooks et politiques qui maintiennent vos agents IA fiables et autonomes - pour **Claude Code** et le **Agents SDK**.

Les agents IA échouent de manière prévisible. Ils exécutent des commandes destructrices, exposent des secrets, dérivent de leur objectif, se retrouvent bloqués dans des boucles, ou poussent directement vers la branche principale. Sans surveillance, les petites défaillances s'enchaînent.

Failproof AI résout ce problème grâce aux **politiques** - des règles qui s'accrochent à chaque appel d'outil de l'agent et peuvent **détecter les défaillances**, **les atténuer** (bloquer, instruire, assainir), et **vous alerter** quand quelque chose nécessite votre attention. Un tableau de bord local vous permet de tout examiner après coup.

Tout s'exécute en local. Aucune donnée ne quitte votre machine.

## Démarrage rapide

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # activer les politiques
failproofai                      # lancer le tableau de bord
```
