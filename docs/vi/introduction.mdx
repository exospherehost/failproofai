Hooks và policies giữ cho các AI agents của bạn hoạt động đáng tin cậy và tự trị - cho **Claude Code**, **OpenAI Codex**, và **Agents SDK**.

Các AI agents thất bại theo những cách có thể dự đoán được. Chúng chạy các lệnh tàn phá, rò rỉ bí mật, lệch khỏi nhiệm vụ, bị mắc kẹt trong vòng lặp, hoặc push trực tiếp lên main. Nếu không được giám sát, những lỗi nhỏ sẽ dẫn đến hậu quả lớn.

Failproof AI giải quyết vấn đề này với **policies** - các quy tắc hook vào từng lệnh công cụ của agent và có thể **phát hiện lỗi**, **giảm thiểu chúng** (block, instruct, sanitize), và **cảnh báo cho bạn** khi cần sự chú ý. Một bảng điều khiển cục bộ cho phép bạn xem lại mọi thứ sau đó.

Mọi thứ chạy cục bộ. Không có dữ liệu nào rời khỏi máy của bạn.

## Bắt đầu nhanh

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```

Xem hướng dẫn [Getting started](/vi/getting-started) để biết toàn bộ hướng dẫn.
