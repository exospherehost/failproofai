Hooks và policies giúp AI agents của bạn hoạt động ổn định và tự động - dành cho **Claude Code** và **Agents SDK**.

AI agents thất bại theo những cách có thể dự đoán được. Chúng chạy các lệnh nguy hiểm, làm lộ bí mật, lạc đề, bị kẹt trong vòng lặp, hoặc push thẳng lên nhánh main. Khi không có người giám sát, những lỗi nhỏ sẽ dần leo thang thành sự cố lớn.

Failproof AI giải quyết vấn đề này bằng **policies** - các quy tắc móc vào từng lệnh gọi công cụ của agent, có khả năng **phát hiện lỗi**, **xử lý chúng** (chặn, hướng dẫn, làm sạch), và **cảnh báo bạn** khi có vấn đề cần chú ý. Một dashboard cục bộ cho phép bạn xem lại toàn bộ hoạt động sau đó.

Mọi thứ đều chạy cục bộ. Không có dữ liệu nào rời khỏi máy của bạn.

## Khởi động nhanh

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # bật policies
failproofai                      # khởi chạy dashboard
```
