## Bảng điều khiển

| Biến | Mô tả |
|----------|-------------|
| `PORT` | Cổng bảng điều khiển (mặc định: `8020`) |
| `CLAUDE_PROJECTS_PATH` | Ghi đè đường dẫn tìm kiếm thư mục dự án Claude Code |
| `FAILPROOFAI_DISABLE_PAGES=policies,projects` | Danh sách các trang bảng điều khiển cần ẩn, phân cách bằng dấu phẩy |
| `FAILPROOFAI_ALLOWED_DEV_ORIGINS` | Các host/IP được phép truy cập tài nguyên dev. Tương tự `--allowed-origins`. |

## Ghi nhật ký

| Biến | Mô tả |
|----------|-------------|
| `FAILPROOFAI_LOG_LEVEL=info\|warn\|error` | Mức độ nhật ký máy chủ (mặc định: `warn`) |
| `FAILPROOFAI_HOOK_LOG_FILE` | Đường dẫn tùy chỉnh cho file nhật ký hook, hoặc `true` để dùng đường dẫn mặc định (`~/.failproofai/logs/hooks.log`) |

## Đo lường từ xa

| Biến | Mô tả |
|----------|-------------|
| `FAILPROOFAI_TELEMETRY_DISABLED=1` | Tắt tính năng thu thập dữ liệu sử dụng ẩn danh |

## LLM (để đánh giá chính sách)

| Biến | Mô tả |
|----------|-------------|
| `FAILPROOFAI_LLM_BASE_URL` | Endpoint API của LLM (mặc định: `https://api.openai.com/v1`) |
| `FAILPROOFAI_LLM_API_KEY` | Khóa API dùng cho các chính sách dựa trên LLM |
| `FAILPROOFAI_LLM_MODEL` | Tên mô hình (mặc định: `gpt-4o-mini`) |
