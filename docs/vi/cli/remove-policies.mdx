---
title: Gỡ cài đặt policies
description: "Xóa các hook entry khỏi settings của Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

Xóa các hook entry của failproofai khỏi `settings.json` của Claude Code.

Aliases: `failproofai p -u`

## Tùy chọn

| Flag | Mô tả |
|------|-------|
| `--scope user` | Xóa khỏi settings toàn cục (mặc định) |
| `--scope project` | Xóa khỏi settings dự án |
| `--scope local` | Xóa khỏi settings cục bộ |
| `--scope all` | Xóa khỏi tất cả các scope cùng lúc |
| `--custom` / `-c` | Xóa `customPoliciesPath` khỏi config |
| `--beta` | Chỉ xóa các beta policies khỏi config |

## Hành vi

- **Không có tên policy** - xóa tất cả hook entry của failproofai khỏi file settings
- **Tên cụ thể** - vô hiệu hóa các policy đó nhưng vẫn giữ hook đã cài đặt

## Ví dụ

```bash
# Xóa tất cả hook ở phạm vi toàn cục
failproofai policies --uninstall

# Vô hiệu hóa một policy cụ thể (giữ lại hook đã cài đặt)
failproofai policies --uninstall block-sudo

# Xóa hook khỏi tất cả các scope
failproofai policies --uninstall --scope all

# Xóa đường dẫn custom policies
failproofai policies --uninstall --custom
```
