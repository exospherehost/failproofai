---
title: Xem phiên làm việc
description: "Khởi chạy bảng điều khiển để duyệt các phiên agent và quản lý chính sách"
---

```bash
failproofai
```

Khởi động bảng điều khiển web tại `http://localhost:8020`.

## Tùy chọn

| Cờ | Mô tả |
|----|-------|
| `--port <number>` | Cổng để lắng nghe (mặc định: `8020`) |
| `--projects-path <path>` | Ghi đè vị trí tìm kiếm các thư mục dự án Claude Code |
| `--allowed-origins <origins>` | Danh sách host/IP được phép truy cập tài nguyên dev, phân tách bằng dấu phẩy |

## Ví dụ

```bash
# Khởi chạy trên cổng khác
failproofai --port 9000

# Sử dụng đường dẫn dự án tùy chỉnh
failproofai --projects-path ~/my-claude-projects
```
