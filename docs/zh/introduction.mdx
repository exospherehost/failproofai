让您的 AI 智能体保持可靠且自主运行的钩子与策略——适用于 **Claude Code** 和 **Agents SDK**。

AI 智能体的失败方式往往有迹可循：执行破坏性命令、泄露密钥、偏离任务、陷入循环，或直接推送到主分支。若无人看管，小问题会不断扩大。

Failproof AI 通过**策略**来解决这些问题——这些规则会介入智能体的每一次工具调用，能够**检测故障**、**进行缓解**（拦截、指令纠正、内容净化），并在需要人工介入时**向您发出警报**。本地仪表板让您可以随时回顾所有操作记录。

所有处理均在本地完成，数据不会离开您的设备。

## 快速开始

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # 启用策略
failproofai                      # 启动仪表板
```

完整操作指南请参阅[入门指南](/zh/getting-started)。
