让您的 AI 智能体保持可靠并自主运行的钩子与策略——支持 **Claude Code**、**OpenAI Codex** 以及 **Agents SDK**。

AI 智能体的失败方式往往是可预见的：执行破坏性命令、泄露密钥、偏离任务、陷入循环，或直接推送到主分支。若无人监管，小问题会不断蔓延。

Failproof AI 通过**策略**来解决这些问题——这些规则会介入智能体的每次工具调用，能够**检测故障**、**加以缓解**（阻止、指示、脱敏）并在需要关注时**提醒您**。本地仪表板让您事后随时查看完整记录。

所有操作均在本地运行，数据不会离开您的设备。

## 快速上手

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # 启用策略
failproofai                      # 启动仪表板
```
