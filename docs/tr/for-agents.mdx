Tam Failproof AI referansını kodlama aracına tek bir komutla ekle. Claude Code, Cursor, Windsurf ve skill desteği olan diğer tüm araçlarla çalışır.

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills`, hangi araçların kurulu olduğunu otomatik olarak algılar ve her biri için uygun formatta skill'i ekler.

## Skill neleri kapsar

| Alan | Kapsam |
|------|--------|
| Policies | Yerleşik policy isimleri, event türleri, parametreler, etkinleştirme/devre dışı bırakma |
| Özel policies | `customPolicies.add()`, eşleşme filtreleri, `allow`/`deny`/`instruct` API'si |
| Context nesnesi | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| Yapılandırma | `policies-config.json` yapısı, scope birleştirme, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, kapsamlar |
| Dashboard | Oturum görüntüleyici, policy aktivitesi, ortam değişkenleri |
| Mimari | Hook handler akışı, çıkış kodları, stdin/stdout sözleşmesi |

## Skill eksiksiz mi?

Mintlify, navigasyondaki tüm sayfalardan `llms.txt` oluşturur. Failproof AI dokümanları tam API'yi kapsar — her policy, seçenek ve örnek dahildir. Eksik bir şey bulursan kaynak `https://docs.befailproof.ai/llms-full.txt` adresindedir.

Belirli bir bağlam için doğrudan ilgili sayfaya bağlantı ver:

```bash
# Yalnızca özel policies API'si
npx skills add https://docs.befailproof.ai/custom-policies

# Yalnızca yerleşik policies
npx skills add https://docs.befailproof.ai/built-in-policies
```
