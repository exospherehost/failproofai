Failproof AI başvurusunun tamamını kodlama ajantnıza tek bir komutla ekleyin. Claude Code, Cursor, Windsurf ve beceri destekleyen diğer tüm ajantslarla çalışır.

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills` kurulu olduğunuz ajantsları algılar ve beceriyi her biri için otomatik olarak doğru formatta ekler.

## Beceri neleri kapsıyor?

| Alan | Nelerin dahil olduğu |
|------|----------------|
| İlkeler (Policies) | Yerleşik ilke adları, olay türleri, parametreler, etkinleştirme/devre dışı bırakma |
| Özel ilkeler (Custom policies) | `customPolicies.add()`, eşleşme filtreleri, `allow`/`deny`/`instruct` API |
| Bağlam nesnesi (Context object) | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| Yapılandırma (Configuration) | `policies-config.json` yapısı, kapsam birleştirme, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, kapsamlar |
| Pano (Dashboard) | Oturum görüntüleyici, ilke aktivitesi, ortam değişkenleri |
| Mimari (Architecture) | Hook yönetici akışı, çıkış kodları, stdin/stdout sözleşmesi |

## Beceri tam mı?

Mintlify, `llms.txt` dosyasını gezintideki tüm sayfalardan oluşturur. Failproof AI belgeleri tam API'yi kapsar - her ilke, seçenek ve örnek dahildir. Eksik olan bir şey bulursanız, kaynak `https://docs.befailproof.ai/llms-full.txt` adresindedir.

Hedefli bağlam için doğrudan belirli bir sayfaya bağlantı verin:

```bash
# Sadece özel ilkeler API'si
npx skills add https://docs.befailproof.ai/custom-policies

# Sadece yerleşik ilkeler
npx skills add https://docs.befailproof.ai/built-in-policies
```
