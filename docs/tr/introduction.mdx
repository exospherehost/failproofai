AI aracılarınızı güvenilir ve otonom olarak çalışan durumda tutan kancalar ve politikalar - **Claude Code**, **OpenAI Codex** ve **Agents SDK** için.

AI aracıları öngörülebilir şekillerde başarısız olur. Yıkıcı komutları çalıştırırlar, sırları sızıtırlar, görevden sapıp dizayndan çıkarlar, döngülere takılırlar veya doğrudan ana dalına gönderirler. Kontrolsüz bırakıldığında, küçük başarısızlıklar baş döndürücü şekilde artarlar.

Failproof AI bunu **politikalar** ile çözer - her aracı araç çağrısında devreye giren ve **hataları tespit eden**, onları **azaltabilen** (engelleme, talimat verme, temizleme) ve **dikkat gerektiğinde sizi uyaran** kurallar. Yerel bir pano, sonrasında her şeyi incelemeni sağlar.

Her şey yerel olarak çalışır. Hiçbir veri bilgisayarınızdan ayrılmaz.

## Hızlı başlangıç

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # politikaları etkinleştir
failproofai                      # panoyu başlat
```
