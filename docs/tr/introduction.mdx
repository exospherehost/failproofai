**Claude Code** ve **Agents SDK** için AI ajanlarınızı güvenilir ve özerk çalışır hâlde tutan hook'lar ve politikalar.

AI ajanları öngörülebilir biçimlerde başarısız olur. Yıkıcı komutlar çalıştırır, gizli bilgileri sızdırır, görevden saparak döngülere girer ya da doğrudan main branch'e push yapar. Gözetimsiz bırakıldığında küçük hatalar zincirleme büyür.

Failproof AI bunu **politikalar** ile çözer — her ajan araç çağrısına bağlanan, **hataları tespit edebilen**, **hafifletebilen** (engelleyerek, yönlendirerek veya temizleyerek) ve dikkat gerektiren durumlarda **sizi uyaran** kurallar. Yerel bir gösterge paneli, her şeyi sonradan incelemenizi sağlar.

Her şey yerel olarak çalışır. Hiçbir veri makinenizden çıkmaz.

## Hızlı başlangıç

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # politikaları etkinleştir
failproofai                      # gösterge panelini başlat
```
