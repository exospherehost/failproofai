AI ajantslarınızı güvenilir ve otonom olarak çalışan tutmak için hook ve politikalar - **Claude Code** ve **Agents SDK** için.

AI ajantsları öngörülebilir şekillerde başarısız olur. Yıkıcı komutlar çalıştırırlar, sırları sızdırırlar, görevden sapmazlar, döngülere takılırlar veya doğrudan main branch'e push yaparlar. Gözetimsiz bırakıldığında, küçük başarısızlıklar kademeli olarak büyür.

Failproof AI bunu **politikalarla** çözer - her ajan araç çağrısına bağlanan kurallar **başarısızlıkları tespit edebilir**, **azaltabilir** (bloke et, talimat ver, temizle) ve **dikkat gerektiğinde sizi uyarabilir**. Yerel bir dashboard, sonrasında her şeyi incelemenizi sağlar.

Her şey yerel olarak çalışır. Hiçbir veri makinenizi terk etmez.

## Hızlı başlangıç

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # politikaları etkinleştir
failproofai                      # dashboard'u başlat
```

Tam açıklaması için [Başlarken](/tr/getting-started) kılavuzunu görün.
