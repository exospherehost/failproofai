---
title: Politikaları yükle
description: "Politikaları etkinleştir ve her agent araç çağrısında çalışmasını sağla"
---

```bash
failproofai policies --install [policy-names...] [options]
```

Yüklü agent CLI'nizin (Claude Code, OpenAI Codex veya GitHub Copilot CLI _(beta)_) ayarlar dosyasına hook girişleri yazar, böylece failproofai araç çağrılarını engeller.

Takma adlar: `failproofai p -i`

## Seçenekler

| Flag | Açıklama |
|------|----------|
| `--cli claude\|codex\|copilot` | Yüklenecek Agent CLI'lar; boşlukla ayrılmış (örn. `--cli claude codex copilot`) veya tekrarlı. Yüklü CLI'ları otomatik olarak algılamak ve sorgulamak için atlayın. |
| `--scope user` | Kullanıcı kapsamlı ayarlar dosyasına yükle (Claude: `~/.claude/settings.json`; Codex: `~/.codex/hooks.json`; Copilot: `~/.copilot/hooks/failproofai.json`). Varsayılan. |
| `--scope project` | Proje kapsamlı ayarlar dosyasına yükle (Claude: `<cwd>/.claude/settings.json`; Codex: `<cwd>/.codex/hooks.json`; Copilot: `<cwd>/.github/hooks/failproofai.json`). |
| `--scope local` | Yalnızca Claude için — `<cwd>/.claude/settings.local.json` dosyasına yükle. Codex ve Copilot'ın `local` kapsamı yoktur. |
| `--custom <path>` / `-c` | Özel hook politikaları içeren JS dosyasının yolu |

## Davranış

- **Policy adı yok** - politika seçmek için etkileşimli bir istem açar
- **Belirli adlar** - bu politikaları etkinleştirir (zaten etkinleştirilen herhangi birine eklenir)
- **`all`** - mevcut tüm politikaları etkinleştirir

Yükleme toplamalı bir işlemdir: `--install` komutunu tekrar çalıştırmak yeni politikaları ekler, mevcut olanları kaldırmaz.

## Örnekler

```bash
# Tüm varsayılan politikaları genel olarak yükle (etkileşimli)
failproofai policies --install

# Geçerli proje için belirli politikaları yükle
failproofai policies --install block-sudo sanitize-api-keys --scope project

# Tüm politikaları bir kez etkinleştir
failproofai policies --install all

# Özel politikalar dosyası ile yükle
failproofai policies --install --custom ./my-policies.js

# OpenAI Codex için yükle (proje kapsamı)
failproofai policies --install --cli codex --scope project

# GitHub Copilot CLI (beta) için geçerli proje kapsamında yükle
failproofai policies --install --cli copilot --scope project

# Üç CLI için de aynı anda yükle
failproofai policies --install --cli claude codex copilot
```

`--custom <path>` sağlandığında, dosya anında doğrulanır - en az bir kez `customPolicies.add()` çağrısı yapmalıdır. Çözümlenmiş yol, `customPoliciesPath` olarak `policies-config.json` dosyasına kaydedilir.
