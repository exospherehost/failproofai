---
title: Oturumları görüntüleyin
description: "Agent oturumlarına göz atmak ve politikaları yönetmek için panoyu başlatın"
---

```bash
failproofai
```

Web panosunu `http://localhost:8020` adresinde başlatır.

## Seçenekler

| Bayrak | Açıklama |
|------|-------------|
| `--port <number>` | Dinlenecek port (varsayılan: `8020`) |
| `--projects-path <path>` | Claude Code proje klasörlerinin bulunduğu konumu geçersiz kıl |
| `--allowed-origins <origins>` | Geliştirici kaynaklarına erişim izni verilen virgülle ayrılmış ana bilgisayarlar/IP'ler |

## Örnekler

```bash
# Farklı bir portta başlat
failproofai --port 9000

# Özel bir proje yolunu kullan
failproofai --projects-path ~/my-claude-projects
```
