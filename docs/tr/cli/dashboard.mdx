---
title: Oturumları görüntüle
description: "Aracı oturumlarını taramak ve politikaları yönetmek için panoyu başlatın"
---

```bash
failproofai
```

Web panosunu `http://localhost:8020` adresinde başlatır.

## Seçenekler

| Bayrak | Açıklama |
|------|-------------|
| `--port <number>` | Dinlenecek port (varsayılan: `8020`) |
| `--allowed-origins <origins>` | Dev kaynaklarına erişim izni verilen virgülle ayrılmış hostlar/IP'ler |

Panoyu varsayılan olmayan bir Claude proje klasörüne yönlendirmek için başlatırken `CLAUDE_PROJECTS_PATH` ortam değişkenini ayarlayın.

## Örnekler

```bash
# Farklı bir portta başlatın
failproofai --port 9000

# Ortam değişkeni aracılığıyla özel bir Claude projeleri yolunu kullanın
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
