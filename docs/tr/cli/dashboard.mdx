---
title: Oturumları görüntüle
description: "Ajan oturumlarına göz atmak ve politikaları yönetmek için gösterge panelini başlatın"
---

```bash
failproofai
```

Web gösterge panelini `http://localhost:8020` adresinde başlatır.

## Seçenekler

| Bayrak | Açıklama |
|--------|----------|
| `--port <number>` | Dinlenecek port (varsayılan: `8020`) |
| `--projects-path <path>` | Claude Code proje klasörlerinin bulunduğu konumu geçersiz kılar |
| `--allowed-origins <origins>` | Geliştirici kaynaklarına erişmesine izin verilen virgülle ayrılmış host/IP listesi |

## Örnekler

```bash
# Farklı bir portta başlat
failproofai --port 9000

# Özel bir proje yolu kullan
failproofai --projects-path ~/my-claude-projects
```
