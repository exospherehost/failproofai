---
title: Ver sessões
description: "Inicie o dashboard para navegar pelas sessões de agentes e gerenciar políticas"
---

```bash
failproofai
```

Inicia o dashboard web em `http://localhost:8020`.

## Opções

| Flag | Descrição |
|------|-----------|
| `--port <number>` | Porta para escutar (padrão: `8020`) |
| `--allowed-origins <origins>` | Hosts/IPs separados por vírgula com permissão de acesso aos recursos de desenvolvimento |

Para apontar o dashboard para uma pasta de projetos Claude diferente da padrão, defina a variável de ambiente `CLAUDE_PROJECTS_PATH` ao iniciar.

## Exemplos

```bash
# Iniciar em uma porta diferente
failproofai --port 9000

# Usar um caminho personalizado para projetos Claude via variável de ambiente
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
