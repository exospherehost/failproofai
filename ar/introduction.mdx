خطافات وسياسات تجعل وكلاءك الذكيين موثوقين ويعملون باستقلالية - لـ **Claude Code** و**Agents SDK**.

يفشل الوكلاء الذكيون بطرق متوقعة. فهم ينفّذون أوامر مدمّرة، ويسرّبون الأسرار، وينحرفون عن المهمة، ويعلقون في حلقات، أو يدفعون التغييرات مباشرةً إلى الفرع الرئيسي. وإذا تُرك الأمر دون رقابة، فإن الأخطاء الصغيرة تتحول إلى كوارث متتالية.

يحلّ Failproof AI هذه المشكلة من خلال **السياسات** - قواعد تتكامل مع كل استدعاء أداة يقوم به الوكيل، وتستطيع **الكشف عن الأعطال** و**التخفيف منها** (الحظر أو التوجيه أو التطهير) و**تنبيهك** عند الحاجة إلى تدخل. تتيح لك لوحة تحكم محلية مراجعة كل شيء لاحقاً.

كل شيء يعمل محلياً. لا تغادر بياناتك جهازك أبداً.

## البداية السريعة

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # تفعيل السياسات
failproofai                      # تشغيل لوحة التحكم
```

راجع دليل [البدء السريع](/ar/getting-started) للاطلاع على الشرح الكامل.
