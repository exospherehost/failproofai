## لوحة التحكم

| المتغير | الوصف |
|----------|-------------|
| `PORT` | منفذ لوحة التحكم (الافتراضي: `8020`) |
| `CLAUDE_PROJECTS_PATH` | تجاوز المسار الذي يتم فيه البحث عن مجلدات مشاريع Claude Code |
| `FAILPROOFAI_DISABLE_PAGES=policies,projects` | صفحات لوحة التحكم المراد إخفاؤها، مفصولة بفواصل |
| `FAILPROOFAI_ALLOWED_DEV_ORIGINS` | المضيفون/عناوين IP المسموح لها بالوصول إلى موارد التطوير. مماثل لـ `--allowed-origins`. |

## التسجيل

| المتغير | الوصف |
|----------|-------------|
| `FAILPROOFAI_LOG_LEVEL=info\|warn\|error` | مستوى تسجيل الخادم (الافتراضي: `warn`) |
| `FAILPROOFAI_HOOK_LOG_FILE` | مسار مخصص لملف سجل الـ hook، أو `true` للمسار الافتراضي (`~/.failproofai/logs/hooks.log`) |

## القياس عن بُعد

| المتغير | الوصف |
|----------|-------------|
| `FAILPROOFAI_TELEMETRY_DISABLED=1` | تعطيل قياس الاستخدام المجهول عن بُعد |

## نموذج اللغة الكبير (لتقييم السياسات)

| المتغير | الوصف |
|----------|-------------|
| `FAILPROOFAI_LLM_BASE_URL` | نقطة نهاية واجهة برمجة تطبيقات نموذج اللغة (الافتراضي: `https://api.openai.com/v1`) |
| `FAILPROOFAI_LLM_API_KEY` | مفتاح API للسياسات المدعومة بنموذج اللغة الكبير |
| `FAILPROOFAI_LLM_MODEL` | اسم النموذج (الافتراضي: `gpt-4o-mini`) |
